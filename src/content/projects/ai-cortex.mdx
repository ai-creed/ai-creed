## what it does

ai-cortex gives fresh agent sessions consistent, fast knowledge about a project without re-scanning the repo each time. It indexes code with tree-sitter, builds a trigram + call-graph cache, and exposes the result through a CLI, a library, and an MCP server.

## why

New agent sessions start cold. Without cortex, they either re-scan (slow, expensive) or skip exploration (lossy). Cortex is the small local service that keeps the knowledge layer warm.

## usage

```bash
cortex scan ./my-project
cortex query "where is auth handled?"
```
