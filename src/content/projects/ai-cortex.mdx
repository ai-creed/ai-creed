## what it does

ai-cortex gives fresh agent sessions consistent, fast knowledge about a project without re-scanning the repo each time. It indexes code with tree-sitter, builds a call graph + trigram cache, and exposes the result through a CLI, a library, and an MCP server.

## why

New agent sessions start cold. Without cortex, they either re-scan (slow, expensive) or skip exploration (lossy). Cortex is the small local service that keeps the knowledge layer warm.

## usage

```bash
ai-cortex index /path/to/repo
ai-cortex rehydrate /path/to/repo
ai-cortex suggest "where is auth handled" /path/to/repo
```

Register as an MCP server once so agents use it automatically:

```bash
claude mcp add -s user ai-cortex -- ai-cortex mcp
```

MCP tools exposed: `rehydrate_project`, `suggest_files`, `suggest_files_deep`, `index_project`, `blast_radius`, `search_history`.

## session history

Agent harnesses compact prior messages as the context window fills. Corrections the user gave, decisions made mid-task, file paths referenced once — all lost. `search_history` recovers them.

cortex captures each session locally (no repo writes, no cloud), compacts it into structured chunks with sentence embeddings, and exposes it through a single MCP tool. Hooks into Claude Code's `PreCompact` and `SessionEnd` events, plus Codex's `UserPromptSubmit` and `Stop` events, so capture is automatic across both agents.

```bash
# install hooks once
ai-cortex history install-hooks

# search past sessions from inside an agent
search_history { query: "what did we decide about the auth flow?" }
# → [session abc · correction · turn 42 · score 0.91]
# → "no, keep the token in httpOnly cookie — we agreed on that"
```

```bash
# manual capture / management
ai-cortex history capture --session <id>
ai-cortex history list
ai-cortex history prune --before 2026-01-01
ai-cortex history off   # disable capture
```

History is stored under `~/.cache/ai-cortex/v1/<repoKey>/history/` — local, per-project, prunable.

## scope

TypeScript / JavaScript only for call-graph extraction (other languages index but yield no graph). Requires Node ≥ 20. Cache is local per-worktree at `~/.cache/ai-cortex/`.
